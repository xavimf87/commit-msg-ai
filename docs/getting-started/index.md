# Getting Started

commit-msg-ai generates commit messages from your staged changes using a local LLM via Ollama.

This guide will walk you through:

1. [Installing Ollama]({{ site.baseurl }}/getting-started/install-ollama/) — the local model runtime
2. [Choosing a model]({{ site.baseurl }}/getting-started/choose-model/) — pick the right model for your machine
3. [Installing commit-msg-ai]({{ site.baseurl }}/getting-started/install-commit-msg-ai/) — the CLI tool
4. [Configuration]({{ site.baseurl }}/getting-started/configuration/) — set your default model and preferences
5. [Usage]({{ site.baseurl }}/getting-started/usage/) — start generating commit messages
